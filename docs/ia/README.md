# Categoria: IA 🤖

## 📋 Visão geral
Nesta categoria ficam os serviços relacionados a **Inteligência Artificial** para uso local:
- **Ollama** (servidor de modelos e API)
- **Open WebUI** (interface web que consome a API do Ollama)

## 📂 Serviços
- **[Ollama](./ollama/ollama.md)**: API local para rodar modelos.
- **[Open WebUI](./open-webui/open-webui.md)**: UI web para conversar com os modelos.

## 🚀 Como subir (dev)
```bash
docker compose -f dev/ia/ollama/docker-compose.yaml up -d
docker compose -f dev/ia/open-webui/docker-compose.yaml up -d
```
